Aplicación Sports-Social-Media

Aplicación móvil multiplataforma que fusiona una Red Social Deportiva con un potente sistema de Estadísticas Personales de Rendimiento.

Ver Demo

Resumen del Proyecto

Sports-Social-Media es una plataforma completa para atletas y aficionados. El Frontend fue desarrollado con Flutter, lo que permite una experiencia de usuario rápida y nativa tanto en dispositivos iOS como Android. Los usuarios pueden registrar sus entrenamientos, ver el progreso de sus amigos y compartir sus logros deportivos en un feed dinámico.

El Backend se construyó con Node.js para manejar la lógica de negocio, la autenticación de usuarios y la gestión de la base de datos PostgreSQL. Esta arquitectura asegura que el procesamiento de las estadísticas deportivas personales (como el ritmo, la distancia y los récords) sea rápido, preciso y escalable, proporcionando un análisis de rendimiento robusto y confiable.
