App de Terapia — React Native

Aplicación móvil iOS/Android desarrollada en React Native que conecta pacientes con terapeutas profesionales. Agenda, videollamada cifrada, chat seguro y seguimiento clínico, todo sin salir de la app.

Ver demo

Resumen del Proyecto

La app crea un entorno integral para atención psicológica remota: registro de pacientes y terapeutas verificados, emparejamiento por especialidad, agenda con recordatorios, chat en tiempo real y videollamadas cifradas para sesiones. Todo ocurre dentro de la app, sin redirecciones a servicios externos.

La arquitectura móvil usa React Native (con o sin Expo), React Navigation y almacenamiento seguro. El backend expone una API REST en Node.js conectada a PostgreSQL para perfiles, historial de sesiones, escalas estandarizadas y facturación. La comunicación síncrona/asíncrona se maneja con WebSocket y un servicio de videollamada con cifrado. Se incluyen notificaciones push, control de acceso por roles (paciente/terapeuta/admin) y consentimiento de datos. El objetivo es ofrecer una experiencia fluida, segura y confiable, optimizada para uso clínico.
